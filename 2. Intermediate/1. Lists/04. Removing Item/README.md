# Removing / Deleting Items

#### We can delete an item from a list using the del statement or the remove() method

## Using del

```py
my_list = [1, 2, 3, 4, 5]

# Deleting the item at index 2
del my_list[2]
print(my_list)
```

## Using remove() method

```py
my_list = [1, 2, 3, 4, 5]

# Removing the value 3 from the list
my_list.remove(3)
print(my_list)
```
