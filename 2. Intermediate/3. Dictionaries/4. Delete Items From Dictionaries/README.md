# Deleting In Dictionaries

### We can delete items from dictionaries using the del statement or the pop() method.

## del Statement

```py
my_dict = {'name': 'John', 'age': 25, 'city': 'New York'}
del my_dict['age']
```

## pop() Method

```py
my_dict = {'name': 'John', 'age': 25, 'city': 'New York'}
my_dict.pop('age')
```

## popitem() Method

```py
my_dict = {'name': 'John', 'age': 25, 'city': 'New York'}
my_dict.popitem()
```

## clear() Method to Remove All Items

```py
my_dict = {'name': 'John', 'age': 25, 'city': 'New York'}
my_dict.clear()
```
